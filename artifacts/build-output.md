---
id: build-output
name: Build Output
format: json
category: cli
tags: [build, compile, typescript, webpack, ci, errors, warnings]
description: >
  Compilation or build results from a project. Captures errors, warnings,
  duration, and produced artifacts. Produced by agents that run build tools
  like tsc, webpack, cargo, go build, or next build.
author: mentiko
version: 1.0
schema:
  type: object
  properties:
    tool:
      type: string
      description: build tool used (tsc, next build, cargo, go build, webpack, gradle, etc.)
    success:
      type: boolean
      description: true if build completed without errors
    errors:
      type: array
      description: compilation errors (empty array if build succeeded)
      items:
        type: object
        properties:
          file:
            type: string
            description: file path relative to repo root
          line:
            type: integer
            minimum: 1
            description: line number of the error
          col:
            type: integer
            minimum: 1
            description: column number of the error
          message:
            type: string
            description: full error message from compiler
          severity:
            type: string
            enum: [error, fatal]
            description: error severity level
          code:
            type: string
            description: compiler error code (e.g. TS2345, E0308)
        required: [file, line, message, severity]
    warnings:
      type: array
      description: non-fatal warnings (may be non-empty even on success)
      items:
        type: object
        properties:
          file:
            type: string
          line:
            type: integer
          message:
            type: string
          code:
            type: string
        required: [file, line, message]
    duration_ms:
      type: integer
      minimum: 0
      description: total build time in milliseconds
    artifacts_produced:
      type: array
      description: output files generated by a successful build
      items:
        type: string
    error_count:
      type: integer
      minimum: 0
    warning_count:
      type: integer
      minimum: 0
  required: [tool, success, errors, warnings, duration_ms, error_count, warning_count]
validation_rules:
  - if success is true, errors array must be empty
  - if success is false, errors array must have at least one entry
  - error_count must equal length of errors array
  - warning_count must equal length of warnings array
  - duration_ms must be >= 0
  - artifacts_produced is typically empty when success is false
related_artifacts:
  - test-results
  - code-changes
  - command-output
---

## success example

```json
{
  "tool": "tsc",
  "success": true,
  "errors": [],
  "warnings": [
    {
      "file": "web/lib/cli-pipe.ts",
      "line": 144,
      "message": "'result' is declared but its value is never read.",
      "code": "TS6133"
    }
  ],
  "duration_ms": 4218,
  "error_count": 0,
  "warning_count": 1,
  "artifacts_produced": [
    "web/.next/server/chunks/app/page.js",
    "web/.next/server/chunks/app/chains/page.js",
    "web/.next/static/css/app/layout.css"
  ]
}
```
