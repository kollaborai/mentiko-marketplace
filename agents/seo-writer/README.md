# SEO Writer

Writes SEO-optimized articles for a given topic and keyword.

## Usage

Trigger with `content_requested` or `article_needed`. Pass your brief as the task:

```
Write a 1200-word article about "how to use tmux for developers".
Target keyword: tmux tutorial
Audience: intermediate developers
```

## Output

- Full article with H1/H2/H3 structure
- Meta description (150-160 chars)
- Keyword density report
- Internal linking placeholders
